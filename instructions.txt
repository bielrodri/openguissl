Este é um pequeno manual de como compilar e executar a aplicação:

1ºPASSO:
ative o virtualenv
. venv/bin/activate

2º PASSO:
navegue até a diretoria do flaskr(onde está "setup.py")

3ªPASSO:
na primeira utilização da aplicação usar o seguinte comando:
pip install -r requirements.txt

3º PASSO:
Logo após os requerimentos serem instalados realize os seguintes comandos:
export FLASK_APP=flaskr.flaskr
pip install --editable .
export FLASK_APP=flaskr            
export FLASK_DEBUG=true
flask run

4º PASSO:
Será gerado um URL onde deves copiar e colar em um browser para poder acessar a aplicação
